# Commit Discipline

- `git add <explicit files>` only — never `git add -A` or `git add .`
- Stage only files modified in the current task — nothing else
- One commit per task
- Get explicit user approval before committing — do not auto-commit

For commit message format and body fields, follow `br-impl.md` §5 (Wrap up).
